# notes

Organize your steps to reach your goals, this command is used to manage notes from the Terminal, it is so easy to use, enjoy it!

## Documentation

```
Usage:	notes [OPTION] [DIR]
DIR:	directory where notes are stored, by default (~/Notes)
OPTION:
	-h	help
	-v	version

```
